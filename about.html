About This Project

RouteSafe Barangay 770

A thesis project developing a walking path recommender system that uses Dijkstra's Algorithm to suggest the safest night-time routes for residents of Barangay 770, Sta. Ana, Manila — based on real crime hotspot data.

How Dijkstra's Algorithm Works Here

The system models Barangay 770's street network as a weighted graph. Each intersection is a node; each street segment is an edge. Weights are based on distance plus a safety penalty for nearby crime incidents.

Weighted Safety Graph

Streets near danger zones receive higher edge weights, making Dijkstra's Algorithm naturally prefer paths that avoid those areas when finding the shortest (safest) path.

1
All street intersections in Brgy 770 are mapped as graph nodes
2
Each street segment gets a weight = distance + crime proximity penalty
3
Dijkstra's finds the minimum-weight path from start to destination
4
The result is the route with the best balance of distance and safety

Project Scope

What this system covers and what it doesn't.

Included

  • Walking route recommendations for Barangay 770
  • Crime hotspot visualization from reported incidents
  • Night-time safety scoring per route
  • Dijkstra's Algorithm path calculation
  • Interactive map with danger zone overlays

Not Included

  • Vehicle or motorcycle routing
  • Real-time crime reporting (planned)
  • Routes outside Barangay 770 boundaries
  • Police dispatch integration
  • CCTV or live surveillance feeds
